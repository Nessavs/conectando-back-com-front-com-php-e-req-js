4 Implementações para que funcione com o banco de dados e back com php:

Página ou seção de Depoimentos
- Apresente a lista de depoimentos através da consulta no banco de dados, ou seja, os depoimentos devem estar
gravados no banco de dados para serem carregados e apresentados nesta página.

Página ou seção para adicionar novos depoimentos
- Implemente um formulário com seja possível criar novos depoimentos, ou seja, um cliente poderá adicionar seu
depoimento através desta nova funcionalidade. Implemente as rotinas necessárias para que o depoimento seja gravado
no banco de dados.

Página ou seção Contato
- Ao exibir o Modal de contatos com os campos ((Nome, Email, Mensagem)), implemente as rotinas necessárias para
que a mensagem de contato seja gravada no banco de dados.

Página Principal
- Catálogo de serviços com foto, preço e opção para solicitar orçamento, podem ser carregados de uma tabela de
serviços no banco de dados
